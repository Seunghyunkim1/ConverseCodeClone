### 기술
- React.js
- JavasSript
- Scss
- HTML, CSS

### 구현 페이지, 컴포넌트
- 메인 페이지 (with React Components)
- Banner 및 Header
- 회원가입 / 로그인
- JSON Web Token을 활용한 access token 발행
- 제품 카테고리
- 사이즈, 색상별 제품 필터링
- 제품 상세보기 (with Responsive Web)
- 회원별 장바구니 등록 및 목록 가져오기
- 컨버스 매장정보 보기 (with NaverMap API)
- AWS 백엔드 API 이용
